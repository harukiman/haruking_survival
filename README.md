# Haruking Survival — 俯瞰サンドボックス・サバイバル

ブラウザで遊べる、俯瞰視点（トップダウン2D）のオープンなサンドボックス・サバイバルゲーム。
探索・採掘・クラフト・建築を、手続き生成された無限に広がる世界で楽しめます。

🎮 **プレイ:** https://harukiman.github.io/haruking_survival/

## 特徴
- 🌍 **無限の手続き生成ワールド** — 海・砂浜・草原・森・石山・雪山。シード値で世界が完全再現
- ⛏️ **採掘＆建築** — 木/岩/鉱石(石炭/鉄/金)を採取、ブロックを設置して自由に建築
- 🔨 **クラフト＆道具** — 作業台・かまどで道具や設備を作成、道具のティアで採掘可能な鉱石が広がる
- ❤️ **サバイバル** — 体力・空腹システム、食料で回復、力尽きてもリスポーン
- 💾 **自動セーブ** — LocalStorage に進行状況を保存（シード＋編集差分のみの軽量保存）
- 📱 **PC / スマホ対応** — キーボード/マウス/ゲームパッド、タッチ操作（D-pad＋ボタン）

## 操作方法
**PC:** `WASD`/矢印=移動、左クリック=採掘、右クリック=設置、`1`-`9`=ホットバー、ホイール=切替、`E`=インベントリ
**スマホ:** 左下=移動、右下=採掘/設置/インベントリ

## 技術
- Vanilla JavaScript + HTML5 Canvas + Web Audio API（**ビルド不要・依存ゼロ**）
- チャンク方式 + オフスクリーンキャッシュで 60fps 描画
- value noise / fBm による決定論的な地形生成
- 画像アセット不使用（タイルは実行時に手続き描画）

## ローカルで動かす
```bash
python3 -m http.server 8000
# http://localhost:8000/ を開く
```
