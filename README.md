# Haruking Survival — 俯瞰サンドボックス・サバイバル

ブラウザで遊べる、俯瞰視点（トップダウン2D）のオープンなサンドボックス・サバイバルゲーム。
探索・採掘・クラフト・建築を、手続き生成された無限に広がる世界で楽しめます。

🎮 **プレイ:** https://harukiman.github.io/haruking_survival/

## ✨ 看板システム：二相世界（光と影）
このゲームの核は**ひとつの座標に重なる2つの世界**。
- 夜の敵から **影の欠片** を集め、**影鏡** を作ると **影の世界** へ「影渡り(F)」できる
- 影の世界は同じ地形の形をしながら**紫の異界**に変貌し、地形・資源・敵が一変
- 影でしか採れない **影晶 / 光素** から最強の **影鋼装備** を鍛える
- ただし影に長居すると **正気度** が削られ、視界が狭まり幻惑される——**光を絶やすな**
- **裂け目の楔** は両世界の同じ場所に同時に存在し、開けば**両世界共通の保管庫**として拠点を繋ぐ
- **狂気の視界**：正気度が低いほど**幻影鉱脈**が見え、掘れる——狂気は危険だが、もう一つの真実への扉
- 影の祭壇で**影の主（ボス）**を召喚し、撃破して**影核のお守り**（正気耐性）を得る
- 各地の**石碑**が、世界が二相に裂けた物語を語る（全9断片）
- **共鳴遺跡**：光の世界で**封印壁**に閉ざされた宝物庫は壊せない——影の世界の同じ場所にある**共鳴核**を砕けば、封印が解けて宝が現れる（二相をまたぐパズル）

## 目標と結末
- 画面下中央の **🎯目標** が、序盤の生存から影の世界の攻略、そして**世界の統合**までを優しく導く（自由気ままに遊ぶのも自由）
- 影核・光素・影晶を集めて **統合の核** を作り使えば、割れた二相が一つに還る**エンディング**へ

## 操作（PC / スマホ / PS4コントローラ）
- **PC**: WASD/矢印=移動、左クリック=採掘/攻撃、右クリック=設置/対話、1-9/ホイール=ホットバー、E=インベントリ、F=影渡り、M=サウンド
- **スマホ**: 左下=移動、右下=採掘/設置/影渡り/袋
- **PS4パッド**: 左スティック/十字=移動、×=採掘/攻撃、□○=設置/対話、△=影渡り、L1/R1=ホットバー、OPTIONS=インベントリ

## 特徴
- ⚔️ **ハクスラ戦利品** — レアリティ4階層＋ランダムなaffix(鋭利な/吸血の/頑強な等)の装備を厳選・収集。敵/宝箱/深層/周回で良質化
- 🎵 **状況連動BGM＋効果音** — 昼/夜/影世界/ボスで変化する手続き生成サウンド
- 📖 **物語** — 世界が光と影に裂けた神話。オープニング導入・石碑9断片・アイテムのフレーバー
- 🌑 **影の深層 / 周回(NG+)** — 奥ほど危険で豊か、クリア後は実績引継ぎで高難度周回
- 🎯 **目標(クエスト)＋エンディング** — 11段階の任意導線と「世界を還す」物語の結末
- 🌗 **二相世界（光と影）** — 上記の看板システム。正気度・影渡り・固有資源・固有敵
- 🌍 **無限の手続き生成ワールド** — 海・砂浜・草原・森・石山・雪山＋砂漠。シード値で世界が完全再現
- ⛏️ **採掘＆建築** — 木/岩/鉱石(石炭/鉄/金)を採取、ブロックや柵・扉・ランタンで自由に建築
- 🔨 **クラフト＆道具** — 作業台・かまど・焚き火で道具/武器/防具/設備を作成、ティア進行
- 🌙 **昼夜サイクル＆照明** — 滑らかな日照変化、たいまつ/焚き火/ランタンの光源、夜は暗闇
- 👾 **モブ＆戦闘** — 昼は動物(うさぎ/鹿/羊)、夜は敵(ゾンビ/スケルトン/クモ/スライム)。剣で戦い、防具で身を守る
- 🌾 **農業** — クワで耕し種をまき、作物を育てて収穫。料理でお腹を満たす
- 📦 **チェスト収納 / 🛏️ 睡眠** — アイテムを保管、ベッドで朝まで眠る
- 🌧️ **天候** — 雨・雪のパーティクル演出
- 📈 **レベル＆XP** — 敵を倒して経験値、レベルアップで最大体力アップ
- ❤️ **サバイバル** — 体力・空腹システム、食料で回復、力尽きてもリスポーン
- 💾 **自動セーブ** — LocalStorage に進行状況を保存（シード＋編集差分の軽量保存）
- 📱 **PC / スマホ対応** — キーボード/マウス/ゲームパッド、タッチ操作（D-pad＋ボタン）

## 操作方法
**PC:** `WASD`/矢印=移動、左クリック=採掘/攻撃、右クリック=設置/対話/使用、`1`-`9`=ホットバー、ホイール=切替、`E`=インベントリ
**スマホ:** 左下=移動、右下=採掘・攻撃/設置・対話/インベントリ
- たいまつ・焚き火・ランタンで夜の灯りを確保
- 剣で戦い、防具を装備、ベッドで朝まで眠る
- クワで畑を作り種をまいて収穫、かまど/焚き火で料理

## 技術
- Vanilla JavaScript + HTML5 Canvas + Web Audio API（**ビルド不要・依存ゼロ**）
- チャンク方式 + オフスクリーンキャッシュで 60fps 描画
- value noise / fBm による決定論的な地形生成
- 画像アセット不使用（タイルは実行時に手続き描画）

## ローカルで動かす
```bash
python3 -m http.server 8000
# http://localhost:8000/ を開く
```
